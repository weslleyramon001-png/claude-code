# Automações Disponíveis — IXC Provedor

> Tarefas que o Claude pode executar via API do IXC mediante solicitação.

## Financeiro
| Automação | Descrição | Status |
|---|---|---|
| Relatório de recebimentos | Recebimentos do dia/semana/mês | 🔴 Aguardando API |
| Lista de inadimplentes | Clientes em atraso com detalhes | 🔴 Aguardando API |
| Geração de boletos em lote | Boletos vencidos sem remessa | 🔴 Aguardando API |

## Atendimento
| Automação | Descrição | Status |
|---|---|---|
| Relatório base de clientes | Total ativo, bloqueado, cancelado | 🔴 Aguardando API |
| Clientes sem contato recente | Clientes sem OS ou contato >90 dias | 🔴 Aguardando API |

## Suporte
| Automação | Descrição | Status |
|---|---|---|
| OS em aberto | Lista todas OS abertas por técnico | 🔴 Aguardando API |
| OS fora do prazo | OS que passaram do SLA | 🔴 Aguardando API |
| Pesquisa de satisfação | Consolidar avaliações do período | 🔴 Aguardando API |

## Fiscal
| Automação | Descrição | Status |
|---|---|---|
| Notas emitidas no mês | Relatório mensal de NFS-e | 🔴 Aguardando API |
| Notas com erro | NFS-e rejeitadas pela prefeitura | 🔴 Aguardando API |

## Estoque
| Automação | Descrição | Status |
|---|---|---|
| Itens em estoque crítico | Abaixo do mínimo configurado | 🔴 Aguardando API |
| Consumo por técnico | Materiais consumidos por OS/técnico | 🔴 Aguardando API |

## Produtos
| Automação | Descrição | Status |
|---|---|---|
| Lista de planos com preços | Todos os planos ativos | 🔴 Aguardando API |
