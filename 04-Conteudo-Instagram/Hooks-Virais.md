## 📌 Como Usar

- **Reels:** use o hook como fala inicial (0-3s)
- **Carrossel:** use como título do slide 1
- **Caption:** use como primeira linha antes do quebra-linha
- **Stories:** use na caixa de texto com fundo escuro

**Fórmula do hook:** Dor/Curiosidade + Especificidade + Promessa implícita
