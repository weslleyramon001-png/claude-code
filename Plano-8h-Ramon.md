## Agenda da Sessão 8h (prioridade máxima)

### BLOCO 1 — Ativar JARBAS (30-45 min)
Você precisa trazer:
- API Key da Anthropic → console.anthropic.com
- API Key do ElevenLabs → elevenlabs.io/app/profile

Passos:
1. `cd jarbas-backend && bash setup.sh` (script interativo — só preencher as keys)
2. `railway login && railway init && railway up`
3. Copiar URL pública do Railway
4. Abrir `jarbas-ui/index.html` → Settings → colar URL
5. Testar: escrever "oi JARBAS" e ouvir a resposta em voz

### BLOCO 2 — Publicar Pack de Planilhas (45-60 min)
1. Criar conta Kiwify → kiwify.com.br (5 min)
2. Criar ZIP das 13 planilhas do Drive
3. Criar produto no Kiwify + upload do ZIP (15 min)
4. Criar capa no Canva 800x800px — template: fundo escuro + "13 Planilhas" (10 min)
5. Publicar → copiar link de vendas
6. Atualizar Landing Page e emails com o link real

### BLOCO 3 — Configurar MailerLite (30 min)
1. Criar conta → mailerlite.com
2. Criar automação com os 7 emails (copiar de `Sequencia-Email-7-Dias.md`)
3. Criar formulário de captura (lead magnet = planilha de Controle de Gastos grátis)
4. Testar sequência

### BLOCO 4 — Primeiro Post no Instagram (20 min)
1. Usar Caption 07 de `Captions-30-Posts.md` (post de lançamento do produto)
2. Criar arte no Canva (carrossel 13 planilhas)
3. Publicar com link na bio
